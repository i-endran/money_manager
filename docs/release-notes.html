📝 Release Notes

Chronological product updates and release highlights.

v1.0 — 2026-03-14

Major polish release with platform-adaptive navigation, full settings/import-export workflow, stronger PIN/biometric reliability, and refreshed iOS/Android branding.

Highlights

  • Platform-adaptive navigation refresh:
    • iOS Liquid Glass tab bar.
    • Android tab refinements with filled icons and improved inactive contrast.
  • Expanded Settings experience:
    • Import/export actions, app preferences, security controls, and cloud-backup handoff.
    • Android export flow now supports both Share and Save to Downloads.
  • Security/stability fixes:
    • Biometric unlock switched to ReactNativeBiometrics.simplePrompt().
    • Android PIN-change crash path hardened (including vibration safeguards + manifest permission).
  • Accounts summary behavior refinements:
    • Summary subtotal sign/color normalization updates.
    • Follow-up tuning for Holding display behavior.
  • Branding updates:
    • New app icon assets for iOS and Android (final white-based style).
